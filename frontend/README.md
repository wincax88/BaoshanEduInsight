## 特性

- :bulb: **TypeScript**: 应用程序级 JavaScript 的语言
- :gem: **优雅美观**：精心设计的用户界面
- :rocket: **最新技术栈**：使用 React/umi 等前端前沿技术开发
- :iphone: **响应式**：针对不同屏幕大小设计

## 使用

安装依赖：

```shell
$ npm install
```

启动开发服务器：

```shell
$ npm run dev
```

构建生产版本：

```shell
$ npm run build
```
